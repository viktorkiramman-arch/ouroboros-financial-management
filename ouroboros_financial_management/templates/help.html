Interactive Tutorial

Follow this order when setting up a personal, family, company, or group workspace.

Start With Workspaces

First-Time Setup Checklist

Complete these in order for the best analyst results.

Create or select a workspace.

Use Personal, Family, Company, or Group depending on what you are managing.

Add members.

For family mode, add breadwinners and dependents. For company/group mode, add employees, departments, or members.

Review categories.

Categories feed dropdowns, budgets, reports, charts, and the analyst.

Add transactions or import CSV.

Positive values are income. Expense entries use the amount field and leave Income unchecked.

Create budgets.

Budgets are monthly caps by category in the active workspace.

Run the analyst.

The Analyst page creates a health score, budget warnings, recurring detection, anomalies, and member load analysis.

Ask Ouroboros Advisor.

Use the Advisor page to ask finance questions about the active workspace. It does not call an external AI provider.

Model a decision.

Use the Planning Toolkit for emergency-fund, debt-payoff, savings-goal, and allocation scenarios.

Use Settings.

Change theme, animations, chart types, reporting currency, display currency, and real-time conversion preferences.

Common Mistakes

Wrong workspaceCheck the sidebar before adding data.
Missing categoriesAdd categories before bulk imports.
No budgetsThe analyst needs budgets for variance checks.
No members linkedFamily analysis improves when transactions are linked to members.

Transactions

Manual entries support category and member dropdowns. The transaction list uses virtualization to keep large local lists responsive.

Currency Conversion

Transactions keep their original amount and currency snapshot. Display currency converts the visible UI when online using the local server as the proxy. Offline mode keeps the app usable.

Ouroboros Advisor

The advisor answers finance questions from the active workspace without sending data to an external AI provider.

Calendar

Calendar shows daily income, expenses, net movement, and transaction summaries for the active workspace.

Reports

Exports are generated locally. Excel export escapes spreadsheet formula prefixes.

Private by Default

Local mode blocks non-loopback requests. Hosted mode requires HTTPS, a strong secret, and a private production database.

CSV Rules

Use one signed amount column or separate debit/credit columns. Positive signed values are income. Negative signed values are expenses. Empty category values use your Rules page.
